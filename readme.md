Использованные нейросети: Codex

Для чего: помощь с написанием и объяснением кода на JavaScript, CSS.  

Как использовалось:
1. Генерация новых функций, drag & drop, анимаций и кликов.
2. Исправление ошибок в уже написанном коде.  
3. Подсказки по адаптивности и оптимизации работы скрипта.
4. Исправление анимаций: подпрыгивание персонажа, движение искр и плавная прокрутка.
   
Пример промпта: 
Исправь ошибки в JS-коде, который реализует интерактивный текст с подсветкой букв и плавной прокруткой.

Процесс работы
1. Планирование интерактивных блоков веб-плаката.  
2. Создание JSS и CSS структуры.  
3. Использование Codex для генерации и исправления JS кода.  
4. Интеграция с визуальными эффектами и адаптивностью.  
5. Тестирование интерактивности на разных разрешениях экрана.
